Countdown timer

This is the project I am currently working on.



This application, helps track the time remaining until various important dates and events.


Users can add new entries such as car inspections, insurance payments, utility bills, taxes, birthdays, or project deadlines to maintain full control over upcoming deadlines and eliminate situations where we forget something important or remember it at the last minute.


The application uses Firebase for data management, utilizing REST API for retrieving, adding, deleting, and modifying user entries.


This allows for effective storage and real-time synchronization of data.



Although the application currently has some small performance issues, and I plan to refactor it using techniques described in Nadia Makarevich's book 'Advanced React'.


This will be an excellent opportunity to practically apply advanced optimization methods and improve performance in React.

Modules used

Moedim

A light-weight, localizable and customizable, styled React calendar component.

Day.js

A lightweight library for handling dates and times, offering a simple interface and compatibility with moment.js, but with a smaller size.

Firebase

A Google platform providing a set of tools for mobile and web applications, including a real-time database, authentication, hosting, and analytics.

Moment.js

A popular library for manipulating, validating, parsing, and displaying dates and times in JavaScript.

Styled-Components

A library for styling components in React applications, allowing you to write CSS in JavaScript, enabling dynamic and modular styling.

My Games API

I developed the My Games API application, which uses
REST API to manage a games database.


The application allows users to add new game titles, delete and modify existing ones, and add game publishers.


Users can sort games and use the implemented search feature to quickly find interesting titles.


The application offers user login and registration functionalities, with new accounts having default entries added.


The project aimed to enhance skills in working with REST API and develop backend and database skills.


It is fully responsive, ensuring optimal performance across various devices.

For handling user authentication and sessions.

Mongoose.js

For interacting with the MongoDB database.
